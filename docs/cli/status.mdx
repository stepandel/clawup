---
title: "status"
description: "Show the status of your deployed agents"
---

Displays the current status of all agents in your team from Pulumi stack outputs.

## Usage

```bash
clawup status [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |
| `-c, --config <name>` | Config name (auto-detected if only one) |

## Examples

```bash
# Show status table
clawup status

# JSON output for scripting
clawup status --json

# Specific config
clawup status -c production
```
