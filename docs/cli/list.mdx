---
title: "list"
description: "List all saved configurations"
---

Lists all saved Agent Army configurations stored in `~/.agent-army/configs/`.

## Usage

```bash
agent-army list [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

## Examples

```bash
# List configs
agent-army list

# JSON output
agent-army list --json
```
