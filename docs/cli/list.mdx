---
title: "list"
description: "List all saved configurations"
---

Lists all saved Clawup configurations stored in `~/.clawup/configs/`.

## Usage

```bash
clawup list [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

## Examples

```bash
# List configs
clawup list

# JSON output
clawup list --json
```
